涉及的vim插件

所有插件的安装、配置、使用方法信息量太大，且官方README有详细说明，本文不再重复阐述

pathogen
此为 vim插件 的管理工具。安装配置方法请查看插件的 README

NerdTree
此插件会生成工程目录树，个人感觉对于Java开发者用处不大。安装配置方法请查看插件README

TagBar
此插件可生成代码结构列表，按F8打开列表。安装配置方法请查看插件的 README 另外，此插件需要安装 ctags

UltiSnips
代码补全插件。安装配置方法请查看插件的 README

vim-commentary
代码注释插件，很有用。安装配置方法请查看插件的 README

dilimitMate
成对生成{} [] ()，个人感觉用处不大。安装配置方法请查看插件的 README

TagHighlight
语法高亮插件，安装即可。

vim-powerline
漂亮的vim状态栏。安装配置方法请查看插件的 README

neocomplcache
又一个自动补全插件，建议安装。安装配置方法请查看插件的 README

快捷键前缀

tmux使用一系列快捷键来进行操作。特别的是，使用tmux快键键的方式是，按下快捷键前缀 -> 松开 -> 按下快捷键。
tmux的快捷键前缀默认为Ctrl+b，简写为C-b。可在tmux配置文件中修改，一般不去改它。

常用快键键

新建 水平 panel：C-b %
新建 纵向 panel：C-b “ (半角双引号)
切换panel：

  1. C-b 上下左右键
  2. C-b o (是字母o，不是阿拉伯数字零，顺时针切换)
  3. C-b q，在 panel号 消失之前选择 panel号 进行切换

两个panle间来回切换：C-b ; (没看错，就是半角分号)
关闭panel：C-b x
关闭当前 session/window/panel：C-d
